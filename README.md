# Decathlon Sales Analysis 2022

## Project Goal
Analyze 2022 sales data to provide actionable insights for retail marketing team (Decathlon dataset). My project demonstrates the full data lifecycle: from raw data cleaning to final visualization.

## What I did
* **Data Cleaning (SQL):** Handled messy raw data by converting French dates, fixing numerical formats with REGEXP, and handling NULL values to ensure data integrity.
* **Analysis (MySQL):** Calculated key metrics like total revenue (€719k), profit (€104k), and identified top customers/sales reps.
* **Visualization (Power BI):** Migrated the initial Excel analysis to Power BI to create a dynamic, interactive dashboard.

## Key Skills Demonstrated
* **SQL:** Data cleaning, Type casting, REGEXP, Aggregate functions.
* **BI:** Power BI, DAX, Dashboard Design.
* **ETL:** Moving data from flat files to a structured SQL database.
